📰 科技动态日报自动化系统

基于N8N工作流构建的智能科技资讯聚合平台，每日自动生成包含AI精选摘要的HTML日报，支持历史回溯浏览。

✨ 核心功能

	•	多源聚合：同步抓取10+权威科技媒体RSS源
	•	AI提炼：Gemini生成结构化摘要与要点分析
	•	自动排版：模板引擎实时生成响应式HTML
	•	版本归档：GitHub Commit历史追溯每日更新
	•	零干预：全自动化工作流每日准时发布

📋 使用指南

	•	日报页面：http://dtd.yanel.me
	•	查看历史：点击导航栏日期选择器
	•	原始数据：查看仓库/archive目录
	•	修改模板：编辑提示词模版
